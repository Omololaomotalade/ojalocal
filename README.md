#OMOLOLA OMOTALADE LOCAL MARKET PLACE

## Overview
A local ecommerce platform built to explore inventory management challenges in Nigeria.

## Key Focus
- FIFO & LIFO inventory issues
- Resource mismanagement
- Data visibility gaps in local markets

## Features
- Product listing
- Add to cart
- Checkout flow

## QA Approach
- Functional testing
- UI/UX testing
- Exploratory testing

## Known Issues
- Checkout UI overlap
- Cart responsiveness issues

## Tools
- Cypress (for automation)
- Manual testing
